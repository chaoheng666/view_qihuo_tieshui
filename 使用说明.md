# 股指期货升贴水监控 - 使用说明

## 系统简介

这是一个用于监控股指期货升贴水的单页看板，覆盖 4 个品种：

- `IF`：沪深 300 指数期货
- `IC`：中证 500 指数期货
- `IH`：上证 50 指数期货
- `IM`：中证 1000 指数期货

系统会统一抓取期货和指数数据，自动计算：

- 升贴水点数
- 升贴水率
- 年化升贴水率
- 30 日均值、标准差、分位数、z-score
- 当日实时曲线与 30 个交易日平滑背景趋势

## 快速启动

### 方式 1：双击启动脚本

直接运行：

```text
启动监控.bat
```

### 方式 2：命令行启动

```bash
python main.py --web --port 5000
```

启动后访问：

```text
http://127.0.0.1:5000
```

## 首次运行

首次运行如果缺少依赖，启动脚本会自动尝试安装：

```bash
python -m pip install -r requirements.txt
```

如果自动安装失败，可以手动执行上面的命令。

## 页面说明

### 1. 顶部四品种摘要卡

- 展示每个品种当前主力合约
- 展示实时升贴水率和升贴水点数
- 展示 30 日分位与 z-score
- 点击卡片可切换当前品种

### 2. 主图

- 前景：当日实时主力合约升贴水率
- 背景：最近 30 个交易日日终升贴水率的 5 日 EMA 平滑趋势
- 横轴：真实时间

### 3. 详细指标

- 主力合约
- 期货价格
- 指数价格
- 升贴水点数
- 升贴水率
- 年化升贴水率
- 成交量
- 持仓量
- 到期天数
- 30 日统计值
- 数据来源与数据质量

### 4. 合约表

- 展示当前品种全部可交易合约
- 连续合约 `IF0 / IC0 / IH0 / IM0` 不进入主表
- 表内显示各合约基差和状态

## 数据来源

### 实时数据

- `IF / IC / IH` 期货：AkShare 主源，新浪期货备源
- `IM` 期货：新浪期货主源，最近有效快照兜底
- 指数：新浪指数主源，东方财富指数备源

### 历史数据

- 指数日线：AkShare 指数日线
- 期货日线：CFFEX 历史日线
- 30 日背景趋势：最近 30 个交易日日终主力升贴水率

## 数据文件

项目根目录下的 `data/` 是唯一数据目录：

- `data/realtime_snapshots.csv`：分钟级实时快照
- `data/daily_main_premium.csv`：交易日日终主力升贴水

## 常见问题

### 启动脚本出现乱码

旧版批处理文件如果保存成错误编码，`cmd` 会把乱码当命令执行。现在启动脚本已经改成纯 ASCII，避免这个问题。

### 页面打开了但没有数据

优先检查：

1. 浏览器开发者工具里是否有前端报错
2. `http://127.0.0.1:5000/api/dashboard/overview` 是否能直接打开
3. 机器是否能访问新浪和 AkShare 所需数据源

### 30 天背景趋势为空

说明历史回填还在进行，或最近历史数据暂时未补齐。系统会自动回填最近 45 个自然日，并取其中最近 30 个交易日。

## 计算口径

```text
升贴水点数 = 期货价格 - 指数价格
升贴水率 = (期货价格 - 指数价格) / 指数价格 * 100%
年化升贴水率 = 升贴水率 * (365 / 到期天数)
```

## 建议

- 交易时段内建议每 20 秒刷新一次
- 关注 `待确认`、`陈旧`、`降级` 这些质量状态
- 升贴水率请结合指数波动、到期时间和成交持仓一起看
